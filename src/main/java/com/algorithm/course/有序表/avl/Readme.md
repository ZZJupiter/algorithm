# AVL树

## 定义

1) 首先是一棵搜索二叉树

2) 任意一个节点的左子树的高度和右子树的高度差小于2

## 特点

对平衡性特别敏感

### 可能同时是LL和LR型,或者同时是RL和RR型,不可能同时是LL和RR型

如同时是LL和LR型:

使用LL型调整总能调整平衡,使用LR型调整不一定调整平衡

如果同时是RL和RR型:

使用RR型调整总能调整平衡,使用RL型调整不一定调整平衡
