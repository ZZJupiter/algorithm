处理时机,将数组中的数字逐个入栈,发现破坏单调性时,逐个出栈,直至重新满足单调性,停止出栈.

* 对每一个出栈的元素,右侧首个小于出栈元素的值,即为即将进栈的值.
* 对每一个出栈的元素,左侧最后一个小于出栈元素的值是,元素出栈后,位于栈顶的元素
